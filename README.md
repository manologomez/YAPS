YAPS
====

Yet Another PDF Signer for .NET

Small library used to sign PDF files using a digital certificate and (optionally)
apply a Timestamp using a RFC 3161 compatible server. It uses iTextSharp which is not included.

See the code in the Yaps.Cmd project for example usage.
